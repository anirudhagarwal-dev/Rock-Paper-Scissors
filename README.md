# Rock Paper Scissors

A simple browser game of Rock–Paper–Scissors built with `HTML`, `CSS`, and `JavaScript`.

## Quick Start
- Open `index.html` in any modern browser.
- Optionally use a local server (e.g., VS Code Live Server) to run it.

## How To Play
- Click an icon for your choice: rock, paper, or scissors.
- The computer picks randomly and the result is shown.
- Scores update after each round.

## Project Structure
- `index.html` – markup and game UI
- `style.css` – styles
- `script.js` – game logic and score handling
- `images/` – icons for the choices

## Tech
- Vanilla `HTML`, `CSS`, and `JavaScript`
